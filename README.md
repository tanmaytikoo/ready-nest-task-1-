FLUX
Build Forms. Capture Data. Stay in Flow.

FLUX is a modern, minimalistic form builder that enables users to create, customize, publish, and manage dynamic forms with ease. Designed with simplicity and efficiency in mind, FLUX provides a clean interface for building forms, collecting responses in real time, and organizing data through an intuitive dashboard.

Features
Secure User Authentication
Dynamic Drag-and-Drop Form Builder
Multiple Field Types
Real-Time Response Collection
Form Publishing & Sharing
Response Management Dashboard
Basic Analytics & Insights
Custom Themes & Branding
Mobile-Friendly Responsive Design
Tech Stack
Frontend: Next.js, TypeScript, Tailwind CSS
Backend: Node.js, Next.js API Routes
Database: PostgreSQL
ORM: Prisma
Authentication: JWT / NextAuth
Design Philosophy
FLUX embraces a monochromatic, distraction-free design language. Every interface element serves a purpose. No unnecessary animations, no clutter, and no fake data. The focus remains on creating forms and managing responses as efficiently as possible.

Goal
To provide a lightweight yet powerful alternative to traditional form-building platforms, delivering a professional SaaS experience with a clean, human-crafted codebase.

FLUX transforms information into structured data, keeping ideas, feedback, and workflows in constant motion.
